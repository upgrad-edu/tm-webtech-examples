This <p> has a margin of 50px. So, basically it is 50px away from all the surrounding elements.

This paragraph has four different margins for the four different sides.

This paragraph has property "margin: 30px 10px 15px 40px". This means the paragraph element has, in a clockwise fashion, 30px for top, 10px for right, 15px for bottom and 40px for the left.

This paragraph has property "margin: 30px 10px 15px". This means the paragraph element has 30px for top, 10px for right and left, and 15px for bottom.

This paragraph has property "margin: 30px 10px". This means the paragraph element has 30px for top and bottom, and 10px for right and left.

This paragraph has property "margin: 30px". This means the paragraph element has 30xp for all the sides.

This div is horizontally centered, because of "margin: auto".

This paragraph inherits its margin value from its parent div.
