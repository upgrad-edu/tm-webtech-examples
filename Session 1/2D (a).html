<div> is a block element. If you inspect this element using Dev Console, you will see it occupies an entire line of the window.

This is a <p> element. This is also a block element. If you inspect this element using Dev Console, you will see it occupies an entire line of the window.

This is a <span> element. This is an inline element. If you inspect this element, you will notice it occupies only the width of its content.
